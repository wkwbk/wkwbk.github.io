# 香港旅游攻略

## 香港概况

**香港特别行政区**（Hong Kong），简称"港"，位于中国南部、珠江口东侧，与深圳一河之隔。被誉为"东方之珠"、"购物天堂"、"美食之都"。

### 基本信息

| 项目 | 信息                             |
| ---- | -------------------------------- |
| 面积 | 约1114平方公里                   |
| 人口 | 约750万                          |
| 语言 | 粤语为主，英语、普通话通用       |
| 货币 | 港币（HKD），1人民币 ≈ 1.1港币   |
| 电压 | 220V，英式三脚插头（需带转换器） |
| 时区 | UTC+8（与内地相同）              |
| 驾驶 | 靠左行驶（与内地相反）           |

### 行政区划

> 18个区，分为香港岛、九龙、新界三大区域

| 区域       | 下辖区                                               | 特色                                   |
| ---------- | ---------------------------------------------------- | -------------------------------------- |
| **香港岛** | 中西区、湾仔、东区、南区                             | 商业中心、维多利亚港、太平山、海洋公园 |
| **九龙**   | 油尖旺、深水埗、九龙城、黄大仙、观塘                 | 购物天堂、庙街夜市、黄大仙祠           |
| **新界**   | 荃湾、屯门、元朗、北区、大埔、沙田、西贡、葵青、离岛 | 自然风光、大屿山、迪士尼、湿地公园     |

### 文化特色

- **中西交融** — 150多年殖民历史，留下了独特的中西混合文化
- **茶餐厅文化** — 港式奶茶、菠萝油、蛋挞是日常灵魂
- **电影之都** — 无数经典港片的取景地，星光大道致敬影坛传奇
- **粤语文化** — 独特的语言与俚语，TVB 剧集的发源地
- **赛马文化** — 每逢赛马日，跑马地马场气氛热烈

## 最佳旅行时间

| 季节 | 月份   | 特点                               |
| ---- | ------ | ---------------------------------- |
| 春季 | 3-5月  | 温暖潮湿，偶有阵雨，适合逛街购物   |
| 夏季 | 6-8月  | 炎热多雨，有台风，但夏季打折力度大 |
| 秋季 | 9-11月 | 天气最舒适，秋高气爽，最佳旅行季节 |
| 冬季 | 12-2月 | 凉爽干燥，圣诞新年气氛浓厚，打折季 |

> **推荐**：10-12月是香港旅游的黄金时段，天气舒适，且赶上各种节庆和打折季。

## 经典路线推荐

### 经典初体验线

**时间**：3-4天

**Day1 维港 → Day2 港岛经典 → Day3 九龙扫街 → Day4 离岛休闲**

适合第一次去香港的旅行者，涵盖最经典的打卡点。

### 深度文化线

**时间**：4-5天

**中环石板街 → 上环古董街 → 西营盘 → 深水埗 → 大澳渔村 → 长洲岛**

适合喜欢人文探索、街拍和老香港风情的旅行者。

### 亲子欢乐线

**时间**：3-4天

**迪士尼乐园 → 海洋公园 → 天际100 → 科学馆 → 星光大道**

适合带小孩的家庭出行。

## 各区域攻略

### 中环 & 半山

**时间**：半天-1天

**必去景点：**

- **太平山顶** — 香港地标，凌霄阁观景台俯瞰维港夜景，山顶缆车体验（单程 HK$62）
- **中环半山扶梯** — 全球最长户外有盖自动扶梯，全长800m
- **石板街（砵典乍街）** — 百年石阶，港片经典取景地
- **兰桂坊** — 酒吧一条街，夜生活圣地
- **文武庙** — 上环百年古庙，香火鼎盛
- **PMQ 元创方** — 旧警察宿舍改造的创意园区

**美食推荐：**

- 兰芳园（丝袜奶茶鼻祖）
- 九记牛腩（米其林推荐，只收现金）
- 胜香园（番茄面、咸柠七）
- 莲香楼/莲香居（传统早茶）

### 湾仔 & 铜锣湾

**时间**：半天-1天

**必去景点：**

- **铜锣湾** — 香港最繁华的购物区，时代广场、SOGO 崇光百货
- **湾仔蓝屋** — 战前唐楼建筑群，了解老香港生活
- **维多利亚公园** — 香港最大公园，春节花市在此举行
- **红磡体育馆（红馆）** — 华语歌手演唱会圣地
- **天后庙** — 铜锣湾地名由来

**美食推荐：**

- 何洪记（云吞面，米其林一星）
- 利强记北角鸡蛋仔
- 公和豆品厂（豆腐花）
- 池记（鲜虾云吞面）

### 尖沙咀

**时间**：半天-1天

**必去景点：**

- **维多利亚港** — 每晚 8:00 的"幻彩咏香江"灯光秀（免费）
- **星光大道** — 李小龙铜像、电影明星手印
- **尖沙咀钟楼** — 百年历史地标
- **海港城** — 香港最大购物中心，700+品牌
- **1881 Heritage** — 旧水警总部改造，欧式建筑拍照打卡
- **K11 MUSEA** — 文化零售地标，建筑本身就是艺术品
- **香港文化中心/太空馆/艺术馆** — 维港边文化建筑群

**美食推荐：**

- 一兰拉面（24小时营业）
- 澳门茶餐厅（猪扒包、厚多士）
- 稻香（港式饮茶连锁）
- 翠华餐厅（冰镇奶茶、鱼蛋粉）

### 旺角 & 油麻地

**时间**：半天-1天

**必去景点：**

- **女人街（通菜街）** — 露天市场，各类小商品
- **波鞋街（花园街）** — 球鞋爱好者天堂
- **庙街夜市** — 香港最有名的夜市，港片常见场景
- **油麻地警署** — 《无间道》取景地，已活化
- **朗豪坊** — 旺角地标商场，全港最长室内扶梯
- **金鱼街** — 整条街都是水族店，非常奇特

**美食推荐：**

- 富记粥品（及第粥）
- 义顺牛奶公司（双皮奶、姜撞奶）
- 麻辣烫/街头鸡蛋仔/咖喱鱼蛋
- 十八座狗仔粉

### 深水埗

**时间**：半天

**必去景点：**

- **鸭寮街** — 电子产品跳蚤市场，淘宝天堂
- **大南街** — 新兴文艺咖啡街区
- **美荷楼青年旅舍** — 公屋活化项目，了解香港基层生活史
- **嘉顿山** — 俯瞰深水埗全景的小众日落点

**美食推荐：**

- 公和荳品廠（豆腐花，始于1893年）
- 坤记煲仔饭
- 合益泰小食（肠粉）
- 刘森记面家（虾子捞面）

### 大屿山 & 离岛

**时间**：1-2天

**必去景点：**

- **香港迪士尼乐园** — 全球最小但最精致的迪士尼之一，门票 HK$639
- **昂坪360缆车** — 25分钟空中旅程，俯瞰大屿山和南海（来回 HK$235）
- **天坛大佛** — 全球最大户外青铜坐佛，免费参观
- **大澳渔村** — "东方威尼斯"，棚屋、咸鱼、虾酱
- **长洲岛** — 太平清醮抢包山（每年5月）、芒果糯米糍
- **南丫岛** — 徒步天堂，榕树湾到索罟湾经典行山路线

**美食推荐：**

- 大澳虾酱/咸蛋黄雪糕
- 长洲芒果糯米糍（甘永泰）
- 长洲大鱼蛋
- 南丫岛海鲜（建兴亚婆豆腐花）

### 南区

**时间**：半天-1天

**必去景点：**

- **海洋公园** — 集海洋馆、机动游戏、大熊猫于一体，门票 HK$498
- **赤柱** — 异国风情小镇，赤柱市场、美利楼
- **浅水湾** — 香港最美海滩之一，免费
- **鹤咀** — 小众徒步路线，雷音洞、蟹洞

## 实用贴士

### 证件准备

- **港澳通行证** — 必备，需提前办理（首次约15个工作日）
- **签注** — 个人旅游签注（G签）或团队旅游签注（L签）
- 签注有效期内可多次往返，每次最多逗留7天

### 通讯上网

- 购买香港本地电话卡（中国移动/3HK/CSL，约 HK$50-100）
- 开通内地手机的港澳流量套餐
- 香港大部分商场和咖啡店有免费 WiFi

### 支付方式

- **八达通** — 交通、便利店、快餐店通用
- **支付宝/微信** — 大部分商场和连锁店支持
- **现金** — 小店、街边摊、茶餐厅可能只收现金，建议带少量港币
- **信用卡** — Visa/Master 通用，银联也普遍接受

### 注意事项

1. **住宿贵且小** — 香港酒店房间普遍很小，提前有心理准备，可选择住深圳每天过关
2. **空调极冷** — 商场、地铁、巴士冷气非常足，随身带一件薄外套
3. **靠左行走** — 扶梯靠右站立、左侧通行；行人靠左走
4. **禁止吸烟** — 室内及部分公共场所全面禁烟，违者罚款 HK$1500
5. **饮水** — 香港自来水可直饮，但习惯喝热水的建议带保温杯
6. **小费** — 餐厅一般会收10%服务费，不强制额外给小费
7. **台风季** — 6-9月注意天文台台风信号，8号风球以上公共交通停运

## 穷游省钱攻略

### 住宿省钱

- **住深圳每天过关** — 深圳酒店只有香港的 1/3~1/5，高铁福田到西九龙仅 14 分钟（约 ¥75），每天来回比住港便宜
- **青年旅舍** — 美荷楼 YHA（深水埗）约 HK$200-300/晚，环境不错还有历史展览
- **住新界/荃湾** — 远离市中心但地铁方便，酒店便宜 30-50%
- **民宿/合住** — 旺角、深水埗一带的合住床位约 HK$100-200/晚

### 平价美食攻略

**深水埗 — 全港最便宜的美食街区：**

- 合益泰肠粉 — HK$10 起
- 公和荳品廠豆腐花 — HK$10
- 坤记煲仔饭 — HK$40-50
- 刘森记虾子捞面 — HK$30-40

**茶餐厅套餐 — 你的好朋友：**

- 常餐/快餐 HK$40-60，包含主食+饮品，管饱管好
- 叉烧饭/烧鹅饭 HK$40-50，分量十足

**街头小吃巡礼（花 HK$50 吃饱）：**

- 鸡蛋仔 HK$15
- 咖喱鱼蛋 HK$15
- 格仔饼 HK$20
- 碗仔翅 HK$15
- 煎酿三宝 HK$15

> **省钱秘诀**：避开尖沙咀、铜锣湾的游客餐厅，去深水埗、北角、土瓜湾等本地人聚集的区域吃饭。

### 交通省钱

- 八达通乘地铁有折扣
- 市中心多**步行**，中环→湾仔→铜锣湾走路即可
- 叮叮车 HK$3 横穿港岛，比地铁便宜又能观光
- 天星小轮 HK$3-5 过维港，比地铁有情调还便宜
