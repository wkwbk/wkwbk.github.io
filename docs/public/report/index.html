技术亮点总结：

1. 性能优化：引入线程池与RabbitMQ，将海量平台数据导入的平均处理耗时降低了 60%。

工作中存在的问题与复盘

发现的问题

● 业务耦合度：初期开发中，部分选择组件与特定模块耦合较紧，导致多处重复代码。

● 数据兼容性：面对不同平台的Excel格式，解析逻辑初期较为脆弱。

改进措施

● 抽象与解耦：抽离出通用的 ObjectSelector 与 usePageRefresh 组合式函数。

● 容错增强：建立 AbstractPlatformDataImporter 模板，引入表头校验机制。
