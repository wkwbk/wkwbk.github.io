# 技术选型指南

| 类型         | 推荐组合                      | 适合项目           |
| ------------ | ----------------------------- | ------------------ |
| 网页应用     | React + Spring Boot / Express | 企业后台、内容平台 |
| 移动应用     | React Native / Flutter        | 移动端 APP 开发    |
| 快速开发网站 | Vue + Django / Laravel        | 中小型项目         |
| SEO 要求网站 | Next.js + 任意后端            | 电商、博客         |
