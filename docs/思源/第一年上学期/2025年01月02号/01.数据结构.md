## 存储结构

1. 线性表存储结构

    ```c
    #define MAX_SIZE 100 // 定义顺序表的最大长度

    typedef int ElemType; // 定义元素类型，这里假设为 int 类型

    typedef struct {
        ElemType elem[MAX_SIZE]; // 线性表占用的数组空间
        int last;              // 记录线性表中最后一个元素在数组中的位置，空表置为 -1
    } SeqList;
    ```

2. 单链表存储结构

    ```c
    typedef struct Node {
        ElemType data;
        struct Node* next; // next 是一个指向 struct Node 类型的指针
    } Node, *LinkList;
    ```

3. 双向链表存储结构

    ```c
    typedef struct DNode {
        ElemType data;
        struct DNode *prior, *next; // 和单链表的区别就是多了一个指针域，一个指向前驱一个指向后继
    } DNode, *DoubleList;
    ```

    双向链表的删除操作：

    int DlinkDel(DoubleList L, int i, ElemType* e) {
        DNode* p;
        *e = p->data;
        p->prior->next = p->next; // 删除一号线，连接二号线 // [!code focus]

    }

4. 循环队列存储结构

    ```c
    #define MAX_SIZE 50 // 队列最大长度

    typedef int QueueElementType; // 定义队列元素类型

    typedef struct {
        QueueElementType element[MAX_SIZE]; // 存储队列元素的数组
        int front; // 队头指针，指向队头元素
        int rear; // 队尾指针，指向队尾元素的下一个位置
        int count;
    } SeqQueue;

    // 初始化操作
    void InitQueue(SeqQueue* Q) {
        Q->front = 0;
        Q->rear = 0; // 初始化时队头和队尾指针重合
    }

    // 获取队列长度
    int GetLength(SeqQueue* Q) {
        return (q->rear - q->front + MAX_SIZE) % MAX_SIZE;
    }
    ```

## 循环队列判空和判满

循环队列的判空和判满通常有两种实现方法。

### 预留一个存储单元

- 核心思想：

  - 队列在存储元素时，总是空出一个存储单元。

  - 判空：`front == rear`

  - 判满：`(rear + 1) % MAX_SIZE == front`

- 优缺点：

  - 优点：实现简单，逻辑清晰。

  - 缺点：浪费一个存储单元，实际可用存储空间为 MAX_SIZE - 1。

- 判空和判满实现：

    ```c
    // 判空：front 和 rear 重合时表示队列为空
    int IsEmpty(CircularQueue *q) {
        return q->front == q->rear;
    }

    // 判满：队尾指针的下一个位置与队头指针重合时表示队列满
    int IsFull(CircularQueue *q) {
        return (q->rear + 1) % MAX_SIZE == q->front;
    }
    ```

### 引入额外变量

- 核心思想：

  - 使用一个变量 `count` 来记录队列中的实际元素个数。

  - 判空：`count == 0`

  - 判满：`count == MAX_SIZE`

- 优缺点：

  - 优点：没有浪费存储单元，队列的存储空间完全利用。

  - 缺点：需要额外的变量，逻辑稍复杂。

- 判空和判满实现：

    ```c
    typedef struct {
        QueueElementType element[MAX_SIZE];
        int front;
        int rear;
        int count; // 计数器，记录队列元素个数
    } SeqQueue;
    ```
