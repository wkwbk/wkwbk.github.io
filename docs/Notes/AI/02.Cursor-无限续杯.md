# Cursor 无限续杯

## 先决条件

- 实现 [科学上网](https://lisir.me/GFW/使用/00.v2rayN-使用教程)
- 注册 [Cloudflare](https://dash.cloudflare.com/sign-up)
- 拥有一个或多个域名，且托管到 Cloudflare
- 免费域名注册商：
  - US.KG：<https://nic.us.kg>
  - EU.org：<https://nic.eu.org>

## 自动注册

1. 将 MX 和 TXT 记录组合添加到 DNS 中

    点击启用电子邮件路由，会自动将记录添加到 DNS 中。

2. 配置 Cloudflare 电子邮件目标地址

    将你的 QQ 邮箱添加到目标地址，Cloudflare 会发送一个验证邮件到你的 QQ 邮件中，点击验证即可。

3. 配置 Cloudflare 电子邮件路由规则

    编辑 Catch-All 地址，操作选择发送到电子邮件，目标位置选择第一步你添加的 QQ 邮箱，保存之后将状态设为活动。

4. IMAP 邮箱配置

    获取 QQ 邮箱授权码或密码。

5. 下载软件

    <https://cloud.lisir.me/Teambition/软件/编程类/Cursor>

6. 编辑配置文件

    ```txt
    # 代理（仅在自动注册时有效）
    BROWSER_PROXY='http://127.0.0.1:10808'    # 提前打开 v2rayN

    # Cloudflare 域名邮箱地址
    DOMAIN=你的域名               # [必填] 域名

    # IMAP 邮箱配置
    IMAP_SERVER=imap.qq.com       # [必填] IMAP 服务器地址
    IMAP_PORT=993                 # [必填] IMAP 端口，SSL 通常为 993
    IMAP_USER=你的 QQ 邮箱        # [必填] 邮箱地址
    IMAP_PASS=你的授权码          # [必填] 邮箱授权码或密码
    IMAP_DIR=inbox                # [可选] 默认为收件箱（inbox）

    # 验证设置
    TURNSTILE_TIMEOUT=20            # Turnstile 验证超时时间（秒）
    VERIFICATION_CODE_TIMEOUT=180   # 验证码邮件有效期（秒），默认 3 分钟
    ```

7. 运行程序即可
