# Java | XML

## 作用

Java XML 是指在 Java 编程语言中处理 XML（Extensible Markup Language）的技术。Java 提供了多种 API 来解析、生成和转换 XML 文档，例如 DOM、SAX 和 StAX。XML 常用于数据交换、配置文件和文档存储。
