# Java | JUL

## 作用

JUL（Java Util Logging）是 Java 核心库中提供的日志 API。它允许开发者在 Java 程序中记录各种级别的日志信息，例如调试信息、错误信息和警告信息。JUL 可以帮助开发者诊断问题、监控程序运行状态和进行性能分析。
