# Word 中英文及符号空格批量删除

要去除 Word 中中文和英文、数字或标点符号之间的空格，最快捷的方法是使用内置的**查找和替换**功能。利用通配符精准定位“中文+空格+符号”或“符号+空格+中文”的组合，并将空格删除。

## 批量删除字符间的空格

### 第一步：开启通配符查找

- **打开替换窗口**：在 Word 中按下快捷键 `Ctrl + H`
- **展开高级选项**：点击对话框左下角的 **更多 >>**（或“高级”）按钮
- **勾选通配符**：在选项列表中，勾选 **使用通配符**

### 第二步：分步进行替换

针对不同类型的字符组合，请分多次进行替换：

| 步骤 | 查找内容 | 替换为 | 说明 |
| :--- | :--- | :--- | :--- |
| **去除“中文与英文”之间的空格** | `([!^1-^127])( )([a-zA-Z])` | `\1\3` | 定位“中文字符 + 空格 + 英文字母” |
| **去除“英文与中文”之间的空格** | `([a-zA-Z])( )([!^1-^127])` | `\1\3` | 定位“英文字母 + 空格 + 中文字符” |
| **去除“中文与数字”之间的空格** | `([!^1-^127])( )([0-9])` | `\1\3` | 定位“中文字符 + 空格 + 数字” |
| **去除“数字与中文”之间的空格** | `([0-9])( )([!^1-^127])` | `\1\3` | 定位“数字 + 空格 + 中文字符” |
| **去除“中文与中文/全角标点”的空格** | `([!^1-^127])( )([!^1-^127])` | `\1\3` | 定位“中文/全角标点 + 空格 + 中文/全角标点” |
| **去除“中文与半角标点”之间的空格** | `([!^1-^127])( )([\,\.\!\?\:\;])` | `\1\3` | 定位“中文字符 + 空格 + 常见半角标点” |
| **去除“半角标点与中文”之间的空格** | `([\,\.\!\?\:\;])( )([!^1-^127])` | `\1\3` | 定位“常见半角标点 + 空格 + 中文字符” |

- **执行替换**：设置完成后，点击 **全部替换**（如果存在多个连续空格，可能需要多点几次全部替换）

## 核心原理说明

::: info 匹配规则解析

- **`[!^1-^127]`**：代表所有的中文字符及全角标点（即非 ASCII 字符）。
- **`( )`**：代表我们要查找并删除的那个空格。
- **`[a-zA-Z]`**：代表所有的英文字母。
- **`[0-9]`**：代表所有的阿拉伯数字。
- **`[\,\.\!\?\:\;]`**：代表常用的英文半角标点符号（如需匹配更多半角符号，可在此括号内按格式继续补充）。
- **`\1\3`**：代表保留原来查找到的第 1 部分和第 3 部分，从而巧妙地把中间的空格剔除。

:::
